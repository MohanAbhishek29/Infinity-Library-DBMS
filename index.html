Library Database Overview

Welcome to the Infinity Library Management System. Use the following tested queries to demonstrate the power of your database during the viva presentation.

1. Basic Operations

SELECT * FROM books; SELECT first_name, email FROM members WHERE membership_type = 'Premium';

2. Aggregate Functions

SELECT category_name, COUNT(*) FROM categories c JOIN books b ON c.category_id = b.category_id GROUP BY category_name;

3. The Virtual View (Reporting)

SELECT * FROM active_borrowers_report;

4. Trigger Verification

(Run an INSERT, then check the audit logs and book copies)

INSERT INTO borrow_records (member_id, book_id, borrow_date, due_date) VALUES (3, 6, '2026-02-27', '2026-03-15'); SELECT * FROM audit_logs;
